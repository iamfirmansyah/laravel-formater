# iamfirmansyah/formater
simple formater for laravel packages

# how to use
- add in file composer.json
 
    "require": {
          "iamfirmansyah/formater": "dev-master"
    },
   
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/iamfirmansyah/laravel-formater.git"
        }
    ]
